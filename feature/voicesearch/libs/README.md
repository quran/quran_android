# Sherpa-ONNX AAR

Place the `sherpa_onnx-release.aar` file in this directory.

## How to obtain the AAR

1. Download the pre-built Android libraries from:
   https://github.com/k2-fsa/sherpa-onnx/releases/download/v1.12.25/sherpa-onnx-v1.12.25-android.tar.bz2

2. Extract and build the AAR following the instructions in:
   https://github.com/k2-fsa/sherpa-onnx/tree/master/android/SherpaOnnxAar

3. Copy the resulting `sherpa_onnx-release.aar` to this `libs/` directory.

The AAR is ~15 MB and contains the native JNI libraries for all Android ABIs.
